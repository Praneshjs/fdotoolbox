FDO Toolbox is an application to perform spatial data management and processing. It is written in C# and uses the Feature Data Objects (FDO) API

FDO Toolbox requires the following components to be already installed.

    * .net Framework 2.0
    * Visual C++ 2005 SP1 Runtime Library

FDO Toolbox is hosted on Google Code at http://fdotoolbox.googlecode.com. You can download the latest binary releases and check out the latest source code from subversion. You can also submit any issues and/or feature enhancement requests via the Issue Tracker.
