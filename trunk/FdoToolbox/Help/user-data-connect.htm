Introduction
Connecting via the Generic dialog

To create a new connection, type connect in the Application Console, or by clicking on any of the following menu items:

  • New - Connection - Custom in the Object Explorer toolbar
  • New Connection - Custom in the FDO Data Sources context menu
  • File - Connect to Data in the main menu

You will get the following dialog as shown here:

From here, do the following:

  1. Choose the desired FDO provider
  2. Enter the values for the connection properties in the grid
  3. Enter the name for the connection
  4. Optionally click Test Connection to test the connection settings
  5. Click Connect to create the connection

Once created, your new connection will appear under the FDO Data Sources node in the Object Explorer

Connecting to SDF/SHP the "express" way

You can create an SDF or SHP connection without needing to use the Generic dialog, by clicking on any of the following menu items:

  • New - Connection - SDF or SHP in the Object Explorer toolbar
  • New Connection - SDF or SHP in the FDO Data Sources context menu

A file picker will be displayed. Select the SDF or SHP file to open and a new connection will be created for that file.

Connection Notes

When connecting to RDBMS-based provider (eg. MySQL), the connection process is usually a two-stage process. The first stage is entering the connection parameters in the generic dialog, this is followed by a follow-up dialog which asks which data-store you want to connect to.
