Introduction

The Object Explorer is the central component of FDO Toolbox. It allows you to administer and manage most aspects of the application.

There are 4 main nodes on the Object Explorer:

  • FDO Data Sources
  • ADO.net Data Sources
  • Tasks
  • Modules

The FDO Data Sources node allows you to create and perform operations on FDO connections to geospatial data

The ADO.net Data Sources node allows you to create and perform operations on ADO.net connections using OLEDB

The Tasks node allows you to create and execute tasks

The Modules node allows you to load and view extension module information

Customising Menus

The menus on the Object Explorer are all customisable, allowing you to add/remove commands to/from the context menus. To customise the menus, open the OEMenuMap.xml file in the FDO Toolbox directory.

Then locate the <ContextMenus> element. From here you can do any of the following.

  • To add a new context menu, add a <ContextMenu> sub-element. This element must have an attachTo attribute defined. The value is the name of the node that this context menu will be attached to.
  • To add a new sub-menu to an existing menu, add a <Menu> sub-element to the desired <ContextMenu> element. The <Menu> must have a name attribute, whose value is the name of the sub-menu.
  • To add a new menu entry to an existing menu, add a <Command> sub-element to the desired <ContextMenu> element. The <Command> must have a name attribute, whose value is the name of the command that will be invoked by this menu entry. You can optionally specify a displayName attribute whose value will override the default text on the new menu entry.
